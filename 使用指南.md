## 密钥功能说明

### 1. 密码安全性

```python
# 注册时：加密密码
password_hash = generate_password_hash(password)
# 结果: $2b$12$...

# 登录时：验证密码
is_correct = check_password_hash(stored_hash, user_password)
# 结果: True/False
```

### 2. 会话管理

```python
# 登录成功时创建会话
session['user_id'] = str(user['_id'])
session['username'] = user['username']
session['email'] = user['email']

# 登出时清空会话
session.clear()
```

### 3. 验证规则

| 字段 | 规则 |
|------|------|
| 用户名 | ≥3 字符，全局唯一 |
| 邮箱 | 包含 @，全局唯一 |
| 密码 | ≥6 字符，bcrypt 加密 |
| 确认密码 | 必须与密码相同 |

## 错误处理

### 注册时可能的错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| 所有字段都是必需的 | 某些字段为空 | 填写所有字段 |
| 用户名至少 3 个字符 | 用户名太短 | 使用至少 3 个字符 |
| 密码至少 6 个字符 | 密码太短 | 使用至少 6 个字符 |
| 两次输入的密码不一致 | 密码不匹配 | 确保两次输入相同 |
| 邮箱已被注册 | 邮箱已存在 | 使用不同的邮箱 |
| 用户名已被占用 | 用户名已存在 | 使用不同的用户名 |

### 登录时可能的错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| 邮箱和密码不能为空 | 字段为空 | 填写所有字段 |
| 邮箱或密码错误 | 验证失败 | 检查邮箱和密码 |
| 数据库连接失败 | MongoDB 离线 | 启动 MongoDB 服务 |

## 故障排除快速参考

### 问题：无法连接数据库

```bash
# 检查 MongoDB 是否运行
mongod

# 验证连接
mongo --eval "db.adminCommand('ping')"
```

### 问题：忘记用户名或密码

```bash
# 在 MongoDB 中查看用户列表
mongo
use station_agent_db
db.users.find({}, {username: 1, email: 1})
```

### 问题：需要删除用户

```bash
mongo
use station_agent_db

# 删除特定用户
db.users.deleteOne({email: "user@example.com"})

# 清空所有用户（谨慎操作）
db.users.deleteMany({})
```
